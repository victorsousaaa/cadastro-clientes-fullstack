# Sistema de Cadastro de Clientes - Full Stack

## 🎯 Sobre o Projeto
Sistema completo para gerenciamento de clientes com validações avançadas, máscaras automáticas e API REST robusta.

## 🚀 Tecnologias Utilizadas
- **Frontend:** Vue.js 3, HTML5, CSS3, JavaScript ES6+
- **Backend:** Go (Golang), Gin Framework, GORM
- **Banco:** PostgreSQL
- **API:** REST com validações duplas

## ✨ Funcionalidades
- ✅ Cadastro com validação de CPF brasileiro
- ✅ Máscaras automáticas (CPF/telefone)
- ✅ Email único no sistema
- ✅ Busca e ordenação em tempo real
- ✅ Edição inline com validações
- ✅ API REST completa (CRUD)
